📌 About the Project

This project monitors a directory in Linux for real-time file changes such as creation, modification, or deletion. When a change is detected, an automatic compressed backup is created using shell scripting.

🚀 Key Features

  • 📂 Real-time file monitoring
  • 🔄 Automatic backup generation
  • 🕒 Timestamped backup files
  • 📝 Activity logging
  • ⚙️ Fully automated using Shell Script

🛠 Technologies Used

Linux OS Bash Scripting inotifywait HTML CSS JavaScript

⚙️ How It Works

  1. User selects a directory to monitor
  2. inotifywait detects file changes
  3. Backup is created automatically
  4. Logs are saved with timestamp








    
🖥 Shell Script Code

    
This Bash script monitors a directory and creates automatic backups.


    
#!/bin/bash

MONITOR_DIR="$HOME/monitor_folder"
BACKUP_DIR="$HOME/backup_folder"
LOG_FILE="$HOME/backup.log"

inotifywait -m -r -e create,modify,delete "$MONITOR_DIR" |
while read path action file
do
    TIMESTAMP=$(date "+%Y-%m-%d_%H-%M-%S")
    BACKUP_NAME="backup_$TIMESTAMP.tar.gz"

    tar -czf "$BACKUP_DIR/$BACKUP_NAME" "$MONITOR_DIR"

    echo "[$TIMESTAMP] $action detected on $file. Backup created." >> "$LOG_FILE"
done
