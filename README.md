# Linux File Monitoring & Auto-Backup System

## 📌 Project Description
This project monitors a directory in Linux for real-time file changes and
automatically creates compressed backups using Shell Scripting.

## ⚙️ Features
- Real-time directory monitoring
- Automatic backup creation
- Timestamped backup files
- Activity logging
- Fully automated using Bash

## 🛠 Technologies Used
- Linux OS
- Bash / Shell Scripting
- inotifywait
- HTML, CSS, JavaScript

## 🖥 How It Works
1. Directory is monitored using inotifywait
2. File changes are detected instantly
3. Backup is created automatically
4. Logs are saved with timestamps
